مرحله ۱: نصب و راه‌اندازی LoopMIDI

LoopMIDI چیست؟
LoopMIDI یک درایور MIDI مجازی رایگان برای ویندوز است که به شما امکان می‌دهد پورت‌های MIDI مجازی ایجاد کنید. این پورت‌ها مانند کابل‌های MIDI مجازی عمل می‌کنند و امکان ارتباط بین نرم‌افزارهای مختلف را فراهم می‌کنند.

دانلود LoopMIDI

مشخصات نرم‌افزار

  • سازنده: Tobias Erichsen
    توسعه‌دهنده معتبر ابزارهای MIDI
  • حجم فایل: حدود 7.8 مگابایت
    سبک و کم‌حجم
  • سیستم عامل: Windows XP/Vista/7/8/10/11
    سازگاری کامل با تمام نسخه‌ها
  • قیمت: رایگان
    بدون محدودیت زمانی یا عملکردی
دانلود LoopMIDI

راهنمای نصب

  1. فایل loopMIDISetup.exe را اجرا کنید
  2. مراحل نصب استاندارد را دنبال کنید
  3. پس از نصب، برنامه را از منوی Start اجرا کنید
  4. روی دکمه + کلیک کنید تا یک پورت جدید ایجاد شود
  5. نام پورت را وارد کنید (مثلاً Orca Port)
  6. برنامه را در System Tray باز بگذارید
نکته مهم: LoopMIDI باید قبل از اجرای Orca و DAW شما در حال اجرا باشد تا پورت‌های مجازی قابل دسترس باشند.

مرحله ۲: نصب Orca

Orca چیست؟
Orca یک زبان برنامه‌نویسی esoteric دوبعدی است که هر حرف الفبا یک عملگر است. حروف کوچک بر اساس Bang عمل می‌کنند و حروف بزرگ هر فریم اجرا می‌شوند. Orca یک سینتزکننده نیست، بلکه محیط لایوکدینگ انعطاف‌پذیری است که MIDI، OSC و UDP را به سیستم‌های صوتی خارجی مانند Ableton، Renoise، VCV Rack یا SuperCollider ارسال می‌کند.

دانلود Orca

دانلود Orca (Desktop) - نسخه Uxn نسخه آنلاین

نصب و اجرا

Windows 
1. دانلود فایل Orca-win.zip (69 MB)
2. استخراج فایل در پوشه دلخواه
3. اجرای فایل Orca.exe
macOS 
1. دانلود فایل Orca-mac.app.zip (64 MB)
2. استخراج و باز کردن فایل
3. انتقال Orca.app به پوشه Applications
4. اجرای برنامه از Launchpad یا Applications
Linux 
npm install -g orca-c
# یا دانلود AppImage
chmod +x Orca-linux-x64.AppImage
./Orca-linux-x64.AppImage
نکته: Orca بر پایه Uxn emulator کار می‌کند. برای جزئیات بیشتر، به راهنمای Uxn مراجعه کنید.

مرحله ۳: آشنایی با رابط کاربری

نمونه محیط Orca

روی خانه‌ها کلیک کنید تا عملگرها را قرار دهید و با اصول دوبعدی Orca آشنا شوید.

توضیح رابط کاربری

رابط Orca یک گرید دوبعدی است که هر سلول آن می‌تواند یک عملگر، عدد یا نقطه خالی (.) باشد. عملگرها با همسایگان خود تعامل می‌کنند و سیگنال‌ها (Bang) در جهت‌های مختلف جریان می‌یابند. نوار ابزار بالا شامل دکمه‌های Play/Pause، Save و Console است. زوم با Ctrl + / - انجام می‌شود. Orca از base-36 برای اعداد استفاده می‌کند (A=10, B=11, ..., Z=35).

کلیدهای میانبر اصلی

کلید عملکرد توضیحات
Ctrl+N New ایجاد پروژه جدید
Ctrl+O Open باز کردن فایل .orca
Ctrl+S Save ذخیره پروژه
Space Play/Pause شروع/توقف پخش و اجرای عملگرها
Ctrl+K Console باز کردن کنسول برای دستورات
Ctrl+G Guide نمایش راهنمای عملگرها
Ctrl+/ Comment کامنت کردن انتخاب‌شده
Ctrl+] Zoom In بزرگنمایی گرید
Ctrl+[ Zoom Out کوچک‌نمایی گرید
Ctrl+I Insert Mode حالت درج بدون جایگزینی

مرحله ۴: عملگرهای اصلی

نکته کلیدی: حروف کوچک (lowercase) بر اساس Bang (سیگنال فوری) عمل می‌کنند، در حالی که حروف بزرگ (uppercase) هر فریم (frame) اجرا می‌شوند. اعداد و نقاط خالی برای فضا و ورودی/خروجی استفاده می‌شوند. Orca از base-36 برای مقادیر عددی استفاده می‌کند (A=10 تا Z=35).

عملگرهای پایه (Universal)

عملگر نام عملکرد مثال
A Add جمع دو ورودی از شمال و غرب 2 3 A → 5 به شرق
B Subtract تفریق غرب از شمال 5 2 B → 3
C Clock ساعت با نرخ (شمال) و مدولو (غرب) 8 C 4 → 0,1,2,3
D Delay ارسال Bang با تأخیر (شمال) D 8 → Bang هر 8 فریم
E East حرکت به شرق یا Bang E E E E
F If Bang اگر ورودی‌ها برابر باشند a b F → Bang اگر a==b
G Generator نوشتن عملگرها با offset x y len G
H Halt توقف عملگر جنوبی H → توقف
I Increment افزایش عملگر جنوبی I 4 → +1 هر 4 فریم
J Jumper خروجی شمالی J val → خروجی شمال
K Konkat خواندن متغیرهای متعدد len K → خواندن len متغیر
L Lesser کوچک‌ترین ورودی a b L → min(a,b)
M Multiply ضرب دو ورودی 3 4 M → C (12)
N North حرکت به شمال N ↑ N
O Read خواندن با offset x y O → خواندن از x,y
P Push نوشتن شرقی len key val P
Q Query خواندن با offset x y len Q
R Random عدد تصادفی min max R → تصادفی
S South حرکت به جنوب S ↓ S
T Track خواندن شرقی key len val T
U Euclid ریتم اقلیدسی step max U → Bang اقلیدسی
V Variable متغیر a V 5 → ذخیره در a
W West حرکت به غرب W ← W
X Write نوشتن با offset x y val X
Y Jymper خروجی غربی Y val → خروجی غرب
Z Lerp انتقال تدریجی rate target Z
* Bang Bang مداوم * * * *
# Comment کامنت خط # کامنت

مرحله ۵: تنظیمات MIDI

اتصال Orca به DAW

1. تنظیمات در Orca

  1. Orca را اجرا کنید و Space را برای تست فشار دهید
  2. Ctrl+K را فشار دهید تا کنسول باز شود
  3. دستورات زیر را وارد کنید:
midi:list     # لیست پورت‌های MIDI
midi:0         # انتخاب پورت اول (LoopMIDI)
BPM:120        # تنظیم سرعت 120 BPM

اگر پورتی نمایش داده نشد، LoopMIDI را بررسی کنید.

2. تنظیمات در Ableton Live

  1. به Options → Preferences → MIDI بروید
  2. در MIDI Ports، پورت "Orca Port" را فعال کنید (Track: On, Sync: Off)
  3. یک MIDI Track جدید ایجاد کنید
  4. در MIDI From، "Orca Port" را انتخاب کنید
  5. یک VST/AU instrument اضافه کنید و تست کنید

3. تنظیمات در FL Studio

  1. به Options → MIDI Settings بروید
  2. پورت "Orca Port" را Enable کنید و Input را تنظیم کنید
  3. در Channel Rack، یک instrument (مثل Sytrus) اضافه کنید
  4. MIDI Input را به پورت Orca تنظیم کنید
  5. کانال MIDI را با عملگر : در Orca تطبیق دهید
مشکل رایج: اگر صدا نمی‌آید، مطمئن شوید DAW در حال ضبط MIDI است و volume instrument بالا باشد. از کنسول Orca برای دیباگ استفاده کنید.

مرحله ۶: مثال‌های عملی

این مثال‌ها را در Orca کپی کنید، Play بزنید و به DAW متصل کنید تا نت‌ها را بشنوید. هر مثال یک الگوی موسیقی ساده ایجاد می‌کند. مثال‌ها بر اساس مستندات رسمی Orca انتخاب شده‌اند.

مثال ۱: ارسال نت MIDI ساده

ارسال نت C3 هر ۸ فریم روی کانال ۰.

simple_midi_note.orca 
D8...          # Bang هر 8 فریم
.:03C          # ارسال C3 روی کانال 0

مثال ۲: ملودی تصادفی

انتخاب تصادفی نت بین A و G هر ۸ فریم.

random_note.orca 
D8.aRG.        # Delay 8, random A-G
.:03D..        # ارسال نت D3 (مثال)

مثال ۳: ملودی последователь

ملودی ۴ نتی (T C A F E) با Clock 1/8.

melody_sequence.orca 
D804TCAFE      # Track 4 نت
.8C4           # شمارنده 0-3 با 1/8
.:03C....      # پخش C3

مثال ۴: ریتم اقلیدسی

ریتم اقلیدسی با ۳ ضربه در ۸ مرحله.

euclidean_rhythm.orca 
C8.....        # Clock 8
.U38...        # Euclid 3 در 8
..:03C.        # خروجی C3

مثال ۵: پیشرفت آکورد

پیشرفت آکورد C Major به A Minor هر ۱۶ فریم.

chord_progression.orca 
D16.....       # Delay 16
.4C2.....      # Clock 0-1 با 4
..4TCEG.AC.E   # Track آکورد CEG و ACE
...:03C...

مثال ۶: شرطی - هر نت دوم

پخش نت در فواصل خاص با If.

conditional_note.orca 
.2I6.......    # Increment به 6 با 2
D646TCAFEDG    # Delay 6, Track نت‌ها
.:03D......    # ارسال D اگر شرط

مرحله ۷: تکنیک‌های پیشرفته

پروژکتورها (Projectors)

پروژکتورها برای ایجاد پویای عملگرها استفاده می‌شوند. O برای خواندن، X برای نوشتن با offset.

projectors_example.orca 
22O....        # خواندن از offset 2,2
.......
..C....        # C خوانده می‌شود
22XE...        # نوشتن E در 2,2
.......
....E..        # E ظاهر می‌شود

متغیرها (Variables)

V برای ذخیره/خواندن. متغیرها باید بالای محل خواندن نوشته شوند.

variables_example.orca 
aV5            # ذخیره 5 در a
....Va         # خواندن a
.....5         # خروجی 5
aV6..Va        # به‌روزرسانی به 6
.....6         # خروجی 6

انیمیشن پروژکتور (ژنراتور ملودی)

انیمیشن پروژکتور برای ایجاد ملودی پویا.

animate_projector.orca 
C...........   # Clock
.B4.........   # Bounce 0-8 با 4
.1XE.........  # نوشتن E با offset
........:03C   # C
........:03D   # D
........:03G   # G

ترفندهای Golf

ترفندهای کوتاه‌نویسی

# Modulo
6I4.         # 6 % 4 = 2

# Uppercase
.dAZ          # d به D

# Lowercase
CM1.          # C به c

# Not Null
.L0.
..F0          # Bang اگر L != 0

این ترفندها برای کدهای فشرده مفیدند.

پیشنهاد: برای الگوهای پیچیده، از ترکیب Random و Modulo برای تنوع استفاده کنید. متغیرها را برای ذخیره patternها به کار ببرید.

مرحله ۸: عیب‌یابی مشکلات رایج

مشکلات MIDI

  • هیچ صدایی نمی‌آید: بررسی کنید LoopMIDI در حال اجرا باشد، پورت در Orca و DAW انتخاب شده، و instrument volume بالا باشد. از midi:list در کنسول استفاده کنید.
  • نت‌ها اشتباه پخش می‌شوند: اکتاو و نت را چک کنید (C=0, D=2, etc.). Velocity را با F (15) تست کنید. base-36 را به خاطر بسپارید (A=10).
  • تأخیر (Latency) بالا: BPM را پایین بیاورید یا buffer size DAW را افزایش دهید. Orca را با BPM:60 تست کنید.

مشکلات Orca

  • عملگرها اجرا نمی‌شوند: مطمئن شوید Play فعال است و هیچ خطای کنسول وجود ندارد. از Guide (Ctrl+G) برای syntax چک استفاده کنید.
  • فایل ذخیره نمی‌شود: مجوزهای پوشه را چک کنید یا از Save As استفاده کنید.
  • کرش در لینوکس: Uxn emulator را به‌روزرسانی کنید یا از نسخه C port (orca-c) استفاده کنید.
  • متغیر خوانده نمی‌شود: مطمئن شوید متغیر بالای محل خواندن نوشته شده است.
کمک بیشتر: به Discord یا Forum مراجعه کنید و log کنسول را به اشتراک بگذارید.
